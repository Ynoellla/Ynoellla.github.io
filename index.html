Tutorial 1 covered use of lists to mark content, creation of navigation lists, and linking files, email addresses, and telephone numbers.

Tutorial 2 covered many topics such as applying colors in CSS, applying font styles, adding selectors, and defining list styles. I found the color aspect the most useful as I have used those skills the most in this course.

Tutorial 3 covered page layouts. I have learned how to choose the correct layout for my website from this chapter. I found it invaluable while designing this website.

Tutorial 4 dealt with graphic design in CSS stylesheets. I found the information about text and box shadows the most helpful as many of the assignments have required their use.

Tutorial 5 taught how to design to account for mobile users. With mobile browsing becoming more and more popular, the ability to allow mobile users to view your webpage is vitally important.

Tutorial 6 taught how to use tables and columns in a webpage. I used both aspects in the creation of my final project. I used columns and rows to list the graded practices I had completed.

Tutorial 7 taught how to design and implement a web form. A websites ability to gather information to notify people of news is vital to keeping a consistent user base.

Tutorial 8 taught how to include pictures, audio, and video in a website. I found this very interesting as I love watching videos online. I found it interesting to see how it was accomplished.

Tutorial 9 taught how to debug a webpage using developer tools in the browser. I found this extremely important as it will help me identify errors in my webpages in the future.

Tutorial 10 taught about if statements. I have learned some python. I understand how useful if statements can be in code so it was a relief to know there were if statements in webdevelopment as well.
